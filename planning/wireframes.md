### Key Elements
- **Top Navigation**: Teams tab is active/highlighted
- **Season Selection**: Same button group as League page
- **Horizontal Divider**: Separates seasons from round filters
- **Round Filters**: Same two-dropdown system
- **Team Search**: Searchable dropdown with autocomplete
- **Add Button**: [+] icon to add searched team to selection
- **Selected Team Pills**: Pills with team name and [X] to remove
- **Team Counter**: Shows "2/5" to indicate selection limit
- **Team Overview Cards**: Individual cards for each selected team showing:
  - Wins, Losses, Draws
  - Goals For/Against
- **Compare Button**: Large, prominent button to proceed to detailed comparison

### User Actions
- Type in search box to find teams
- Click [+] to add team from dropdown
- Click [Ã—] on pill to remove team from selection
- Select season and round range (filters apply to overview cards)
- Click "Compare Statistics" button to navigate to Team Comparison Page

### Key Elements
- **Selected Teams Display**: Pills showing currently compared teams (can remove/modify)
- **Filter Summary**: Shows active season and round range
- **Tab Navigation**: Four statistical category tabs (General/Attacking/Defensive/Cumulative)
- **Statistical Charts**: Multiple chart types based on category:
  - **General**: Possession, passes, fouls, discipline
  - **Attacking**: Goals, shots, accuracy, conversion rates
  - **Defensive**: Goals conceded, tackles, clearances, blocks
  - **Cumulative**: Rolling averages, form calculations, efficiency metrics
- **Back Navigation**: Button to return to team selection
- **Export Option**: Allow users to export comparison data

### User Actions
- Switch between statistical categories using tabs
- Remove teams from comparison via [Ã—] on pills
- Click back to modify team selection
- Export comparison data
- Hover over charts for detailed tooltips

## Responsive Considerations

All wireframes are designed for desktop-first but should adapt for mobile:

- **Landing Page**: Stack league cards vertically on mobile
- **League Overview**: Make table horizontally scrollable, stack charts vertically
- **Team Selection**: Single column layout for team overview cards
- **Team Comparison**: Tabs become dropdown selector, charts stack vertically

## Design Notes

- **Color Scheme**: Blue/green for primary actions, red for deletions
- **Typography**: Clear hierarchy with larger fonts for headers
- **Spacing**: Generous whitespace between sections
- **Interactive Elements**: Clear hover states and visual feedback
- **Data Visualization**: Consistent chart styling using Recharts library
- **Accessibility**: High contrast, keyboard navigable, screen reader friendly
