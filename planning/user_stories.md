# User Stories

## User Roles

1. **Casual Fan** - A soccer enthusiast who wants to understand team performance and follow their favorite clubs
2. **Fantasy Football Player** - Someone who plays fantasy soccer and needs detailed statistics to make informed team selections
3. **Sports Analyst** - A data-driven user who wants to perform deep-dive analysis on team performance trends

---

## User Stories

### Casual Fan Stories

1. As a casual fan, I want to view the current league standings for the Premier League, so that I can see how my favorite team is performing relative to others in the table.

2. As a casual fan, I want to select a specific season from previous years, so that I can reminisce about how my team performed in championship-winning seasons or understand historical context.

3. As a casual fan, I want to compare my favorite team against their biggest rival, so that I can see which team has better statistics this season and settle debates with friends.

4. As a casual fan, I want to see basic statistics like goals scored and wins in an easy-to-understand format, so that I don't feel overwhelmed by complex analytics.

5. As a casual fan, I want to filter matches to see how my team performed in the second half of the season, so that I can understand if they had a strong finish or faded down the stretch.

### Fantasy Football Player Stories

6. As a fantasy football player, I want to view attacking statistics (goals, shots, shot accuracy) for multiple teams simultaneously, so that I can identify which teams' players are most likely to score points.

7. As a fantasy football player, I want to see defensive statistics like tackles and interceptions, so that I can evaluate which defenders and defensive midfielders are performing well for my fantasy team selection.

8. As a fantasy football player, I want to filter statistics by specific round ranges (e.g., rounds 10-15), so that I can identify teams and players in good recent form rather than relying on full-season averages.

9. As a fantasy football player, I want to see cumulative statistics and rolling averages, so that I can spot performance trends and make decisions based on momentum rather than one-off performances.

10. As a fantasy football player, I want to compare up to 5 teams at once, so that I can efficiently evaluate multiple player options from different clubs when building my fantasy lineup.

### Sports Analyst Stories

11. As a sports analyst, I want to access derived performance metrics like team form components and efficiency ratios, so that I can perform sophisticated analysis beyond basic counting statistics.

12. As a sports analyst, I want to view possession efficiency and conversion rates, so that I can evaluate how effectively teams are using their opportunities and ball control.

13. As a sports analyst, I want to toggle between general, attacking, defensive, and cumulative statistical views, so that I can analyze teams from multiple tactical perspectives in a single comparison.

14. As a sports analyst, I want to examine historical data across multiple seasons, so that I can identify long-term trends and year-over-year improvements in team performance.

15. As a sports analyst, I want to see trailing averages and form calculations automatically updated based on my filtering selections, so that I can quickly assess team momentum within any specified time frame.

---

## Additional User Stories

### Navigation & Usability

16. As any user, I want to easily navigate between different leagues (Premier League, La Liga, Bundesliga, Serie A, Ligue 1), so that I can analyze teams from multiple competitions without confusion.

17. As any user, I want to search for teams using a dropdown with search functionality, so that I can quickly find specific teams without scrolling through long lists.

18. As any user, I want to remove teams from my comparison view by clicking an X icon on their pill, so that I can refine my analysis and focus on the most relevant teams.

### Data & Filtering

19. As any user, I want to specify both a start round and end round when filtering data, so that I can analyze performance across any custom range of matches that interests me.

20. As any user, I want the application to automatically calculate and display derived statistics based on my filtering criteria, so that I always see accurate, up-to-date metrics without manual calculation.

## User Journey Examples

### Journey 1: Casual Fan Checking Team Performance
1. User lands on home page and clicks on "Premier League"
2. User sees league table and notices their team (Arsenal) is in 2nd place
3. User clicks on "Teams" tab and searches for "Arsenal"
4. User views Arsenal's win/loss record and goals scored
5. User adds "Manchester City" for comparison
6. User toggles to "Attacking" stats to see which team has better goal-scoring metrics

### Journey 2: Fantasy Player Making Transfer Decision
1. User navigates to Premier League > Teams
2. User adds 5 teams (Liverpool, Arsenal, Manchester City, Tottenham, Chelsea)
3. User selects current season "2023/24"
4. User sets round filter to "rounds 25-30" (last 6 matches)
5. User toggles to "Cumulative" tab to see rolling averages
6. User analyzes goals per game and shots on target to decide which team's attackers to bring into fantasy squad
7. User removes teams one by one to narrow down decision

### Journey 3: Analyst Studying Tactical Trends
1. User selects Bundesliga league
2. User searches and selects "Bayern Munich" and "Borussia Dortmund"
3. User selects season "2022/23"
4. User sets round range from 1 to 17 (first half of season)
5. User examines defensive statistics and possession efficiency
6. User changes round range to 18-34 (second half)
7. User compares first-half vs second-half metrics to identify tactical adjustments
8. User toggles to "Cumulative" tab to see team form components and efficiency trends
